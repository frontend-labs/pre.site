# FrontEnd-Labs PRE

Environment used with docker & jekyll

**For Linux users**

    ~ docker-compose up

**For OSX users**

    ~ bash upxmacos.sh

If you have problems with start the docker virtual machine, you can reload the process like this

    ~ bash upxmacos.sh --reload
